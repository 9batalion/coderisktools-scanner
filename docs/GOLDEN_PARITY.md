## Contract

Each case records:

- deterministic case ID;
- relative filepath;
- synthetic line/context fixture;
- exact expected legacy detector IDs;
- redacted evidence form;
- fixture provenance.

The corpus uses reserved synthetic values and public example domains. It does
not contain production credentials or raw credential material.
