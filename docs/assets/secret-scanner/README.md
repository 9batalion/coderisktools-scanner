# CodeRiskTools Secret Scanner branding

Brand assets for the public CodeRiskTools Secret Scanner repository.

## Files

- `coderisktools-secret-scanner-banner-dark.png` — dark-background horizontal banner for dark README themes.
- `coderisktools-secret-scanner-banner-light.png` — light-background horizontal banner.
- `coderisktools-secret-scanner-banner-monochrome.png` — black-and-white horizontal variant.
- `coderisktools-secret-scanner-icon.png` — square shield/robot icon for avatars and compact UI.

The supplied artwork is raster PNG on solid backgrounds. The checkerboard-background source was not treated as transparency because the checkerboard is baked into the JPEG; this avoids publishing a misleading “transparent” asset.

## Suggested README usage

```markdown
![CodeRiskTools Secret Scanner](docs/assets/secret-scanner/coderisktools-secret-scanner-banner-dark.png)
```
